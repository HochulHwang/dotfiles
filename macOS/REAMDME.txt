Cheat Sheet

1. Tmux
    0. General
        o tmux new-session -s 'name'
        o tmux ls
        o tmux attach -t 'name'
        o leader + ctrl + t
        o leader + ctrl + c

2. Vim
    0. General
        o leader + s
        o leader + ci cy cs
        o F3 : grep
        o F5 : delete trailing error
        o F8 : paste mode
        o F9 : tagbar
    1. C/C++
        o ctrl + p
        o ctrl + ]
    2. Python
        o , + d g
        o source PATH/TO/VIRTUALENV/bin activate, deactivate ( virtual env )
        o source activate CONDA/VIRTUALENV, source deactivate ( conda )
    3. Latex
        o , + ll
        o , + lv
        o , + lt
    4. Text
        o Table Mode
          - , + tm
          - |, ||
        o Vim-surround
          - ysiw + 'paren'
          - cs 'from' 'to'
          - ds'paren'
          - yss'paren'
          - Visual + S
    5. Git
        o Gstatus
        o -
        o Gcommit -m 'message'
        o Gpush origin master

3. Terminal Record
    1. asciinema rec
    2. asciinema2gif --speed 2 http://~~/api/asciicasts/~~~
